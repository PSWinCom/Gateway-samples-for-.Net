PSWinCom SMS Gateway Code Samples for .Net

This repo contains several code samples that can be used when implementing the PSWinCom SMS Gateway.
You will see how to send SMS messages, and how to receive delivery reports or mobile originated SMS messages.
You will need a PSWinCom Gateway message account to try the samples. A free demo/trial account can be ordered from our product pages, see links below.
